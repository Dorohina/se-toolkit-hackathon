## How to Contribute

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Write/update tests
5. Submit a Pull Request

### Code Style

We follow PEP 8 and use Black for formatting and Flake8 for linting.

```bash
pip install black flake8
black .
flake8 .
```

### Running Tests

```bash
pip install -r requirements-test.txt
pytest tests/ -v
```
